# 결과물 템플릿 — 상세

> meta-dialogue SKILL.md에서 참조

---

대화가 충분히 진행된 후, 다음 중 하나를 제안합니다:

## 1. 아티클 소재 정리

```markdown
## 잠재 아티클: [제목 제안]

### 핵심 인사이트
- 발견한 주요 깨달음 1-3개

### 타겟 독자
- 이 글이 공명할 사람

### 예상 구조
1. 도입: 공감 포인트
2. 전개: 발견 과정
3. 결론: 인사이트 또는 질문
```

## 2. 자기 이해 요약

```markdown
## 이번 대화에서 발견한 것

### 나에 대한 새로운 이해
- ...

### 앞으로 시도해볼 것
- ...

### 남은 질문
- ...
```

## 3. 다음 대화 주제 제안

```markdown
## 더 탐색할 수 있는 방향

1. [주제 A] - 이걸 탐색하면 ~ 알 수 있을 듯
2. [주제 B] - 이 각도로 보면 ~
```
